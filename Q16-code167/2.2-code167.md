## 代码实现

方法1: 首尾指针  时间复杂度:O(n); 空间复杂度: O(1)

```ts
function twoSum(numbers: number[], target: number): number[] {
  let l = 0, r = numbers.length - 1;
  while (l < r) {
    const val = numbers[l] + numbers[r];
    if (val === target) return [l + 1, r + 1];
    val < target ? l++ : r--;
  }
}
```
